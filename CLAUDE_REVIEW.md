# VideoSaaS 代码审查规范

## 安全审查
- 是否引入新权限？是否最小化？
- 文件路径处理是否安全？禁止 `..` 路径穿越
- 敏感信息（API Key、Webhook URL）是否通过环境变量/配置文件注入？
- 输入验证：用户输入的文件路径是否经过校验？

## 架构一致性
- 是否遵循项目现有架构？
- 错误处理是否符合静默降级原则？

## 错误处理
- 所有文件操作必须有 try/except 或异常处理
- 用户提示是否清晰（网络错误、权限不足、路径不存在）

## 性能
- 是否有内存泄漏？
- 大文件处理是否有截断机制？

## 可维护性
- 代码是否清晰易懂？是否添加必要注释？
- Python 版本兼容性（>= 3.9）
